   :class: floatingflask

Introduction
============

This python module implements a sampler for amino acid and small molecule protonation states and tautomers in OpenMM.

It has the following requirements:

* python 2.7, 3.4 or 3.5
* openmm 7.0rc1
* numpy >=1.10
* scipy >=0.17.0
* openmmtools 0.7.5
* pymbar
* openmoltools 0.7.0
* ambermini
* joblib
* lxml
